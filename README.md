# Cattle vs. Buffalo Classifier

[![Python](https://img.shields.io/badge/Python-3.9%2B-blue?logo=python&logoColor=white)](https://www.python.org/)
[![TensorFlow](https://img.shields.io/badge/TensorFlow-2.x-FF6F00?logo=tensorflow&logoColor=white)](https://www.tensorflow.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A binary image classifier that distinguishes **cattle** from **buffalo** using transfer learning on top of **MobileNetV2** (pretrained on ImageNet). Built with TensorFlow/Keras.

## How It Works

1. **Data augmentation** — random horizontal flips, rotation, and zoom are applied during training to improve generalization from a small dataset.
2. **Transfer learning** — a frozen `MobileNetV2` (ImageNet weights, no top layer) is used as a feature extractor.
3. **Classification head** — `GlobalAveragePooling2D → Dropout(0.3) → Dense(1, sigmoid)` sits on top for binary classification.
4. **Training** — the model is trained with the Adam optimizer and binary cross-entropy loss on an 80/20 train/validation split.
5. **Inference** — a saved model can classify any new image as `cattle` or `buffalo` with a confidence score.

## Getting Started

### Prerequisites

- Python 3.9+
- pip

### Installation

```bash
git clone https://github.com/APEKSHAHUDALI/cattle-buffalo-classifier.git
cd cattle-buffalo-classifier

python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate

pip install -r requirements.txt
```

### Train the model

```bash
python train.py
```

Optional flags:

```bash
python train.py --data-dir data --epochs 15 --batch-size 16 --model-out models/cattle_vs_buffalo_model.keras
```

This trains on the images in `data/cattle/` and `data/buffalo/`, prints validation accuracy after each epoch, and saves the trained model to `models/cattle_vs_buffalo_model.keras`.

### Classify an image

```bash
python predict.py --image data/test_image/wild-buffalo-wester-ghats-kerala-shot-inside-forest-196281912.webp
```

Or run without arguments to be prompted for a path interactively:

```bash
python predict.py
```

Example output:

```
Prediction: this image is a BUFFALO
Confidence: 94.32%
```

## Dataset

The `data/` folder contains a small, manually curated set of cattle and buffalo images (~18 cattle, ~27 buffalo) organized into class subfolders, which `tf.keras.utils.image_dataset_from_directory` expects. For better accuracy, expand the dataset with more diverse, higher-resolution images per class before retraining.

## Notes & Limitations

- The current dataset is small; validation accuracy will vary between runs and is not a reliable estimate of real-world performance. Expect it to improve substantially with more training images.
- The base `MobileNetV2` model is frozen (not fine-tuned) — unfreezing the top layers and fine-tuning with a low learning rate is a natural next step for higher accuracy.
- Trained model files (`*.keras`) are not committed to the repository (see `.gitignore`) since they're regenerated by `train.py`.
